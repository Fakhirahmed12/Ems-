# EMS Electric Billing System

EMS Electric Billing System is a billing application that calculates electricity bills based on the number of units consumed. It also generates a printable bill form, making the billing process simple, fast, and efficient.

## Features

- Unit-based bill calculation.
- Automatic bill generation.
- Printable bill form/receipt.
- Clean and simple user interface.
- Fast and accurate billing process.

## Tech Stack

- HTML
- CSS
- JavaScript
- C++ / Backend (if used)

## Usage

- Enter customer details.
- Add the number of units consumed.
- Generate the bill automatically.
- Print the bill form if needed.
